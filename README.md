# ClawProject 🦞

**Seu gestor de projetos com IA, integrado ao OpenClaw.**

O ClawProject ajuda você a sair de uma ideia e chegar em uma entrega real.
Sem complicação: criar projeto, planejar com IA, executar e concluir.

---

## Para quem é

- Gestores de projeto
- Fundadores/operadores
- Times enxutos

---

## O que o MVP já faz

1. Criar projeto pelo dashboard
2. Conversar com o gestor IA do projeto
3. Organizar planejamento em `docs/PLANNING.md`
4. Acompanhar execução em cards
5. Gerar entrega final em `docs/DELIVERY.md`

---

## Fluxo principal

**Create → Plan → Execute → Deliver**

---

## Primeiros 5 minutos (visão de usuário)

1. **Abrir o ClawProject** no navegador (`http://127.0.0.1:19192`)
2. **Clicar em “Criar Projeto”** e preencher nome + objetivo
3. **Conversar com o gestor IA** para refinar o escopo
4. **Clicar em “Iniciar Projeto”** para organizar o roadmap e as tarefas
5. **Concluir com “Deliver”** para gerar o resumo final da entrega

> Obs.: o foco aqui é uso do produto. Detalhes técnicos ficam para documentação de contribuição.
